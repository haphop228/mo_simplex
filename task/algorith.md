
# Двухфазный модифицированный симплекс-метод и конструктивное доказательство теоремы двойственности

## Часть 1. Теоретическая справка

### 1.1. Общая постановка задачи ЛП

В общем виде задача линейного программирования может содержать:
* **ограничения-неравенства** трёх типов: $\leq,\ \geq,\ =$;
* **переменные** с произвольными нижней/верхней границами (включая отрицательные и свободные);
* направление оптимизации $\max$ или $\min$.

**Прямая задача (общая форма):**
$$f(x) = c^T x \to \max\quad (\text{или } \min)$$
$$a_i^T x \,\square\, b_i,\quad \square \in \{\leq,\ \geq,\ =\},\ i = 1, \dots, m$$
$$l_j \leq x_j \leq u_j$$

**Двойственная задача** для прямой $c^T x \to \max,\ Ax = b,\ x \geq 0$:
$$g(u) = b^T u \to \min,\quad u^T A \geq c^T,\quad u\text{ — свободные при }Ax = b.$$

### 1.2. Каноническая форма

Каноническая (стандартная) форма, которой требует симплекс-метод:
$$f(x) = c^T x \to \max,\quad Ax = b,\quad x \geq 0,\quad b \geq 0.$$

Любая общая задача приводится к канонической последовательностью преобразований:

| Преобразование | Действие |
|---|---|
| $\min c^T x$ | заменяется на $\max (-c)^T x$, ответ затем умножается на $-1$ |
| $a_i^T x \leq b_i$ | добавляем балансовую переменную: $a_i^T x + s_i = b_i,\ s_i \geq 0$ |
| $a_i^T x \geq b_i$ | вводим избыточную: $a_i^T x - s_i = b_i,\ s_i \geq 0$ |
| $a_i^T x = b_i$ | без изменений |
| $b_i < 0$ | умножаем строку на $-1$ (знак $\square$ инвертируется) |
| $x_j \leq 0$ | замена $x_j' = -x_j,\ x_j' \geq 0$ |
| $x_j$ свободна | замена $x_j = x_j^+ - x_j^-,\ x_j^+, x_j^- \geq 0$ |
| $x_j \geq l_j,\ l_j \neq 0$ | замена $x_j' = x_j - l_j$, правая часть: $b \leftarrow b - A_j l_j$ |
| $x_j \leq u_j$ | добавляем строку $x_j \leq u_j$ как обычное неравенство |

### 1.3. Зачем нужны две фазы

После приведения к каноническому виду $Ax = b,\ x \geq 0$ нужен **начальный допустимый базис** $B$ — такой, что $B^{-1} b \geq 0$.

* Если в каждой строке $i$ есть «свой» столбец-орт (например, балансовая $s_i$ из неравенства $\leq$ с $b_i \geq 0$), то начальный базис — единичная матрица $E$.
* Если же есть строки с `=` или `≥` (после приведения $b$ к неотрицательным), то столбцов-ортов нехватает. Тогда применяется **метод искусственного базиса**:

> Для каждой «проблемной» строки добавляется **искусственная переменная** $y_i \geq 0$ с коэффициентом $+1$. Получается расширенная система $Ax + Ey = b$ с очевидным базисом $\{y_1, \dots, y_k\}$. Чтобы вытеснить $y_i$ из базиса, решается **вспомогательная (фаза I)** задача:
> $$w(x, y) = \sum_{i \in I_y} y_i \to \min$$
> Если $w_{\min} > 0$ — исходная задача **несовместна**.
> Если $w_{\min} = 0$ — все $y_i$ равны нулю, базис, в котором они остались (если остались), вырожден, и можно перейти к **фазе II** на исходной целевой функции $c^T x$.

### 1.4. Модифицированный (матричный) симплекс-метод

Отличие от табличного: на каждом шаге пересчитывается не вся матрица условий, а только **обратная базисная матрица** $B$ (далее везде под $B$ понимается именно $B^{-1}$ в традиционном обозначении — следуя педагогическому соглашению методички, без степени).

В процессе решения прямой задачи на каждом шаге генерируется вектор $u_0$. В момент, когда прямая задача решена, этот вектор автоматически становится решением **двойственной задачи** (конструктивное доказательство Первой теоремы двойственности).

### 1.5. Словарь переменных (глоссарий)
* $N = (N_1, \dots, N_m)$ — упорядоченный список индексов базисных переменных.
* $B$ — обратная базисная матрица (для текущего $N$).
* $x_B$ — значения базисных переменных, $x_B = B \cdot b$.
* $x$ — **полный** вектор плана размерности $n$: на позициях $N_i$ стоит $x_B[i]$, на остальных — нули.
* $c_B$ — коэффициенты целевой функции при базисных переменных текущей фазы.
* $u_0 = c_B^T B$ — текущий вектор двойственных оценок.
* $\Delta_j = u_0 A_j - c_j$ — оценка небазисной переменной $j$.
* $z_0 = B \cdot A_{j_0}$ — направляющий вектор (полной длины $n$, если разлагать по полному списку столбцов; на практике используется только его базисная компонента длины $m$).
* $t_0$ — длина шага (величина, на которую вводимая переменная увеличится).
* $s_0$ — индекс выводимой из базиса переменной.

## Часть 2. Алгоритм по шагам

### Шаг 0. Приведение к канонической форме и начальный базис

1. Если задача на $\min$ — заменяем $c \leftarrow -c$, и в конце меняем знак ответа.
2. Все ограничения с $b_i < 0$ умножаем на $-1$.
3. Применяем переменные-замены из таблицы 1.2 (свободные, отрицательные, сдвиги границ). Сохраняем «карту перевода» для восстановления исходных переменных в финальном ответе.
4. Для каждой строки определяем тип:
   * `<=` → добавить балансовую $s_i$. Она войдёт в начальный базис.
   * `>=` → добавить избыточную $s_i$ (с коэффициентом $-1$) **и** искусственную $y_i$ (с $+1$). В начальный базис войдёт $y_i$.
   * `=`  → добавить только искусственную $y_i$. В начальный базис войдёт $y_i$.
5. Множество индексов искусственных переменных обозначим $I_y$.
6. Начальный базис $N$ — список переменных, идущих с коэффициентом $+1$ как «своя единичная колонка» (по одной на строку). $B = E$, $x_B = b$.

### Шаг 1 (фаза I). Вспомогательная задача

Если $I_y \neq \varnothing$, выполняется фаза I:

* Целевые коэффициенты фазы I: $\hat c_j = -1$ при $j \in I_y$, иначе $0$ (минимизация суммы $y$ эквивалентна максимизации $-\sum y$).
* Запускаем итерации модифицированного симплекс-метода (см. Шаг 2) на $\hat c$.
* По окончании:
  * **$w_{\min} > 0$** ⇒ задача несовместна, алгоритм завершается с этим вердиктом.
  * **$w_{\min} = 0$**:
    * Если в $N$ не осталось индексов из $I_y$ — переходим к фазе II.
    * Если осталась $y_i$ в базисе с $x_B[i] = 0$ — пробуем вытеснить её обычной pivot-операцией: ищем небазисный $j \notin I_y$ с $z_0[i] \neq 0$ для столбца $A_j$ и заменяем $y_i \to x_j$. Если такого нет — строка линейно зависима, удаляем её (вместе с искусственной из дальнейшего рассмотрения).
* Все искусственные столбцы при переходе к фазе II «замораживаются»: они удаляются из числа кандидатов на ввод в базис (либо им присваивается $\hat c_j = -\infty$, что эквивалентно вечному запрету ввода).

### Шаг 2 (универсальный шаг симплекса). Проверка оптимальности

1. Обновить $x_B = B \cdot b,\ c_B = (c_{N_1}, \dots, c_{N_m})$.
2. Вычислить $u_0 = c_B^T B$.
3. Перебирая небазисные индексы $j$ **по возрастанию**, считать $\Delta_j = u_0 A_j - c_j$:
   * как только встретили $\Delta_{j_0} < 0$ — **немедленно прерываем перебор**, переменная $x_{j_0}$ вводится в базис (правило **первого индекса** / Бленда — гарантирует отсутствие зацикливания);
   * если перебор завершился без $\Delta_j < 0$ — план оптимален, алгоритм текущей фазы завершён.

### Шаг 3. Выбор переменной на вывод (направляющий вектор)

1. $z_0 = B \cdot A_{j_0}$.
2. Если все компоненты $z_0 \leq 0$ (в фазе II) — целевая функция не ограничена.
3. По всем $i$, где $z_0[i] > 0$, считаем $\rho_i = x_B[i] / z_0[i]$.
4. $t_0 = \min_i \rho_i$. Среди индексов, на которых достигается минимум, выбираем тот, у которого $N_i$ минимален (правило Бленда).
5. Переменная $x_{N_{s_0}}$ выводится из базиса.

### Шаг 4. Пересчёт плана и обратной матрицы

1. $N_{s_0} \leftarrow j_0$.
2. Новые значения базисных компонент:
   * $x_B^{new}[s_0] = t_0$,
   * $x_B^{new}[i] = x_B[i] - t_0 \cdot z_0[i]$ для $i \neq s_0$.
3. Пересчёт $B$ методом Гаусса-Жордана с ведущим элементом $\pi = z_0[s_0]$:
   * **Шаг (a):** строка $s_0$ матрицы $B$ делится на $\pi$.
   * **Шаг (b):** для всех $i \neq s_0$: $B[i,*] \leftarrow B[i,*] - z_0[i] \cdot B[s_0,*]$.
4. Возврат к Шагу 2.

### Шаг 5. Восстановление ответа

После завершения фазы II:
1. Полный вектор $x \in \mathbb{R}^{n_{\text{ext}}}$ собирается из $x_B$ и нулей.
2. По «карте перевода» (см. шаг 0.3) восстанавливаются пользовательские переменные исходной задачи.
3. Значение целевой функции: $f^* = c_B^T x_B$ (если задача была на $\min$ — меняем знак).
4. **Решение двойственной задачи** — вектор $u_0$ финального шага фазы II.

## Часть 3. Подробный разбор примера

### 3.1. Постановка (пример «прямой»)
$$ 12x_1 + 3x_2 \to \max $$
$$ 4x_1 + x_2 \leq 16 $$
$$ 2x_1 + 2x_2 \leq 22 $$
$$ 6x_1 + 3x_2 \leq 36 $$
$$ x_1, x_2 \geq 0 $$

Все ограничения $\leq$, $b \geq 0$, искусственные не нужны → фаза I пропускается, сразу фаза II.

### Каноническая форма (балансовые $x_3, x_4, x_5 \geq 0$)
$c = (12, 3, 0, 0, 0)^T$, $b = (16, 22, 36)^T$.
$$ 4x_1 + x_2 + x_3 = 16,\ \ 2x_1 + 2x_2 + x_4 = 22,\ \ 6x_1 + 3x_2 + x_5 = 36 $$

### Шаг 0
$N = (3, 4, 5)$, $B = E$, $x_B = (16, 22, 36)^T$, полный $x = (0, 0, 16, 22, 36)^T$.

### Итерация 1
$u_0 = (0,0,0)\cdot E = (0,0,0)$. Перебираем небазисные **по возрастанию** $j$:
$$\Delta_1 = u_0 A_1 - c_1 = 0 - 12 = -12 < 0 \Rightarrow j_0 = 1.$$
**Перебор прерван** (по правилу первого индекса), $\Delta_2$ не считаем.

$z_0 = E \cdot (4, 2, 6)^T = (4, 2, 6)^T$.
$t_0 = \min(16/4,\ 22/2,\ 36/6) = \min(4, 11, 6) = 4$, минимум на строке 1 (соответствует $x_3$, $N_1 = 3$).
$s_0 = 1$, выводим $x_3$.

### Итерация 2
$N = (1, 4, 5)$, новый $x_B = (4, 14, 12)^T$, полный $x = (4, 0, 0, 14, 12)^T$.
$$B = \begin{bmatrix} 1/4 & 0 & 0 \\ -1/2 & 1 & 0 \\ -3/2 & 0 & 1 \end{bmatrix}.$$
$c_B = (12, 0, 0)$, $u_0 = (12, 0, 0)\cdot B = (3, 0, 0)$.

Перебираем небазисные $j$ по возрастанию:
* $\Delta_2 = u_0 A_2 - c_2 = 3\cdot 1 + 0\cdot 2 + 0\cdot 3 - 3 = 0$. Условие $\Delta_2 < 0$ не нарушено.
* $\Delta_3 = u_0 A_3 - c_3 = 3 - 0 = 3 \geq 0$.

Все $\Delta_j \geq 0$ — **план оптимален**.

### 3.2. Пример с двухфазным методом
$$ x_1 + x_2 \to \max,\quad x_1 + x_2 \leq 10,\quad x_1 + x_2 \geq 10,\quad x_{1,2} \geq 0. $$

Каноническая форма ($s_1$ — балансовая для $\leq$, $s_2$ — избыточная для $\geq$, $y_1$ — искусственная):
$$ x_1 + x_2 + s_1 = 10,\quad x_1 + x_2 - s_2 + y_1 = 10. $$
Начальный базис $N = (s_1, y_1) = (3, 5)$. $B = E$, $I_y = \{5\}$.

**Фаза I.** $\hat c = (0, 0, 0, 0, -1)$, минимизируем $y_1$.
* $\hat c_B = (0, -1)$, $u_0 = (0, -1)\cdot E = (0, -1)$.
* $\Delta_1 = u_0 A_1 - \hat c_1 = (0)(1) + (-1)(1) - 0 = -1 < 0 \Rightarrow j_0 = 1$.
* $z_0 = (1, 1)^T$, $t_0 = \min(10/1, 10/1) = 10$ на обеих строках; по Бленду берём $s_0$ с минимальным $N_{s_0}$, т.е. $s_0 = 1$ (выводим $s_1$).
* После пересчёта: $N = (1, 5)$, $x_B = (10, 0)^T$. $y_1 = 0$ — фаза I завершена с нулём, но $y_1$ осталась в базисе.

Вытесняем $y_1$ из базиса: ищем небазисный $j \in \{2, 3, 4\}$ с ненулевой компонентой $z_0[2]$ нового столбца. Например, $A_2 = (1, 1)^T$, после преобразования $z_0[2] \neq 0$ — заменяем $y_1$ на $x_2$ (или ставим $s_2$, в зависимости от пересчёта). После этого $I_y$ опустошён.

**Фаза II.** $c = (1, 1, 0, 0)$ (искусственный столбец удалён). Запускаем те же итерации; система определяет, что любая точка $x_1 + x_2 = 10$ — оптимальна, $f^* = 10$.

## Часть 4. Выводы и доказательство теоремы двойственности

1. **Решение прямой задачи** — финальный полный вектор $x^*$ из последнего шага фазы II.
2. **Решение двойственной задачи** — вектор $u^*$, равный $u_0$ на этом же шаге.
3. **Сильная двойственность:** $f^*_{\max} = c^T x^* = c_B^T B b = u_0^T b = b^T u^* = g^*_{\min}$.
4. **Условия совместности:**
   * Если фаза I заканчивается с $w_{\min} > 0$ — прямая задача **несовместна** (двойственная неограничена).
   * Если в фазе II найден столбец с $\Delta_{j_0} < 0$ и $z_0 \leq 0$ — прямая задача **неограничена** (двойственная несовместна).

Алгоритм, реализующий обе фазы и возвращающий вектор $u_0$, является **конструктивным доказательством Первой (основной) теоремы двойственности** для общей задачи ЛП.
