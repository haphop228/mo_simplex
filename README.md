# Решатель задач линейного программирования (Двухфазный модифицированный симплекс-метод)

Это приложение решает задачи линейного программирования с пошаговым выводом хода
решения на экран, используя точные арифметические вычисления (без погрешностей
округления).

## Функционал
- Задание произвольного количества переменных и ограничений (`<=`, `>=`, `=`).
- **Двухфазный** симплекс-метод с автоматическим построением искусственного
  базиса (фаза I) и переходом к оптимизации исходной целевой функции (фаза II).
- Корректное определение трёх исходов: оптимум, **несовместность** (фаза I с
  $w_{\min} > 0$), **неограниченность** (фаза II с $z_0 \leq 0$).
- Правило выбора $j_0$ — **первый отрицательный $\Delta_j$** (правило Бленда),
  гарантирует отсутствие зацикливания на вырожденных задачах.
- Пошаговый вывод: на каждом шаге показываются базис $N$, обратная базисная
  матрица $B$, полный вектор плана $x$, базисная подвыборка $x_B$, вектор
  двойственных оценок $u_0 = c_B^T B$, посчитанные оценки $\Delta_j$ (только
  до первого отрицательного), направляющий вектор $z_0$, шаг $t_0$ и
  выводимая переменная.
- Красивое оформление с математическими формулами на базе LaTeX (KaTeX).
- Подсветка фаз: «Фаза I (вспомогательная задача)» / «Фаза II».
- Возможность экспорта пошагового решения в формат Markdown.
- Экспорт на печать (в PDF формата A4) прямо из интерфейса.

## Математическая основа
Полное описание алгоритма (с разбором двух примеров и условиями двойственности)
находится в [`task/algorith.md`](task/algorith.md).

Краткое резюме шагов:
1. **Шаг 0.** Приведение к канонической форме $Ax = b,\ x \geq 0,\ b \geq 0$.
   Для `<=` добавляется балансовая переменная (в начальный базис), для `>=` —
   избыточная и **искусственная** (в начальный базис), для `=` — только
   искусственная.
2. **Фаза I.** Если есть искусственные переменные, минимизируется их сумма.
   - $w_{\min} > 0$ → задача **несовместна**, останов.
   - $w_{\min} = 0$ → переходим к фазе II (искусственные «замораживаются» —
     запрещены к вводу обратно в базис).
3. **Фаза II.** Решается исходная задача $c^T x \to \max$ (минимизация
   приводится через $c \leftarrow -c$).
4. **Выходные данные:** оптимальный $x^*$, значение $f^*$ и вектор двойственных
   оценок $u^* = u_0$ (конструктивное доказательство Первой теоремы
   двойственности).

## Установка и локальный запуск

Для разработки и запуска вам понадобится `Python 3.10+`.

1. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Как скомпилировать независимый исполняемый файл

Сборка осуществляется с помощью `PyInstaller`.
Приложение использует системный браузерный движок для отображения UI
(`pywebview`), поэтому его можно скомпилировать в лёгкий статический бинарник.

### Для Windows (`.exe` файл)
*Важно: компилировать необходимо именно на компьютере под управлением Windows.*
1. Откройте терминал (Command Prompt или PowerShell).
2. Выполните установку:
   ```cmd
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Выполните сборку:
   ```cmd
   python build.py
   ```
4. В папке `dist` появится файл `SimplexSolver.exe`. Вы можете переносить его
   на любую другую систему Windows, он не требует установки Python!

### Для macOS (`.app` пакет)
*Важно: компилировать нужно на Mac.*
1. В терминале выполните установку (см. пункт «Установка и локальный запуск»).
2. Выполните команду сборки:
   ```bash
   python build.py
   ```
3. В папке `dist` появится `SimplexSolver.app`, который можно перенести в папку
   Applications.

### Для Linux
*Важно: компилировать нужно на Linux.*
1. В терминале выполните:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. В Linux для `pywebview` требуются системные библиотеки WebKit (например,
   `libgtk-3-dev`, `libwebkit2gtk-4.0-dev`). Убедитесь, что они установлены.
3. Выполните сборку:
   ```bash
   python build.py
   ```
4. В папке `dist` будет лежать бинарник `SimplexSolver`.

## Тестирование
Для проверки работоспособности математического ядра, запустите:
```bash
export PYTHONPATH=$(pwd)
pytest tests/test_solver.py tests/test_exporter.py tests/test_two_phase.py
```

Тестовый набор:
- [`tests/test_solver.py`](tests/test_solver.py) — базовые проверки солвера.
- [`tests/test_exporter.py`](tests/test_exporter.py) — рендеринг Markdown / HTML.
- [`tests/test_two_phase.py`](tests/test_two_phase.py) — комплексные кейсы:
  3 эталонные задачи (с `>=`, `=`, смешанные), несовместность,
  неограниченность, проверка правила первого индекса, сильной двойственности
  и длины полного вектора $x$.
