# Решатель задач линейного программирования (Модифицированный Симплекс-Метод)

Это приложение решает задачи линейного программирования с пошаговым выводом хода решения на экран, используя точные арифметические вычисления (без погрешностей округления).

## Функционал
- Задание произвольного количества переменных и ограничений (<=, >=, =).
- Пошаговое решение симплекс-методом.
- Красивое оформление с математическими формулами на базе LaTeX (KaTeX).
- Возможность экспорта пошагового решения в формат Markdown.
- Экспорт на печать (в PDF формата А4) прямо из интерфейса.

## Установка и локальный запуск

Для разработки и запуска вам понадобится `Python 3.10+`.

1. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Как скомпилировать независимый исполняемый файл

Сборка осуществляется с помощью `PyInstaller`.
Приложение использует системный браузерный движок для отображения UI (`pywebview`), поэтому его можно скомпилировать в легкий статический бинарник.

### Для Windows (`.exe` файл)
*Важно: компилировать необходимо именно на компьютере под управлением Windows.*
1. Откройте терминал (Command Prompt или PowerShell).
2. Выполните установку:
   ```cmd
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Выполните сборку:
   ```cmd
   python build.py
   ```
4. В папке `dist` появится файл `SimplexSolver.exe`. Вы можете переносить его на любую другую систему Windows, он не требует установки Python!

### Для macOS (`.app` пакет)
*Важно: компилировать нужно на Mac.*
1. В терминале выполните установку (см. пункт "Установка и локальный запуск").
2. Выполните команду сборки:
   ```bash
   python build.py
   ```
3. В папке `dist` появится `SimplexSolver.app`, который можно перенести в папку Applications.

### Для Linux
*Важно: компилировать нужно на Linux.*
1. В терминале выполните:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. В Linux для `pywebview` требуются системные библиотеки WebKit (например, `libgtk-3-dev`, `libwebkit2gtk-4.0-dev`). Убедитесь, что они установлены.
3. Выполните сборку:
   ```bash
   python build.py
   ```
4. В папке `dist` будет лежать бинарник `SimplexSolver`.
